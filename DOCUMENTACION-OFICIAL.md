# Documentacion de referencia para SQL

SQL no pertenece a un solo motor, asi que conviene estudiar el lenguaje con
referencias de motores reales y documentacion bien mantenida.

## Referencias principales

- [PostgreSQL documentation](https://www.postgresql.org/docs/)
- [SQLite documentation](https://www.sqlite.org/docs.html)
- [MySQL reference manual](https://dev.mysql.com/doc/)

## Recursos clave para estudiar

- [PostgreSQL tutorial SQL](https://www.postgresql.org/docs/current/tutorial-sql.html)
- [Referencia SELECT en PostgreSQL](https://www.postgresql.org/docs/current/sql-select.html)
- [SQLite language reference](https://www.sqlite.org/lang.html)
- [Tipos de datos en SQLite](https://www.sqlite.org/datatype3.html)
- [Funciones SQL en MySQL](https://dev.mysql.com/doc/refman/en/functions.html)

## Orden recomendado

1. Aprende SELECT, WHERE, ORDER BY y LIMIT.
2. Pasa a INSERT, UPDATE y DELETE.
3. Estudia joins, agrupaciones y subconsultas.
4. Luego practica modelado, indices y optimizacion.
